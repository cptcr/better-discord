Better Discord Theme

Download Better-Discord.js

Installation Instructions

  1. Download the CSS file using the button above.
  2. Open Better Discord settings in Discord.
  3. Navigate to the Themes section.
  4. Click on the Folder Icon button and when it opens the file manager put in the better-discord.theme.css file.
  5. Apply the theme and enjoy the mobile-inspired UI on desktop!

Installation Instructions

  1. Download the JSON file using the button above.
  2. Open Vencord settings in Discord.
  3. Navigate to the Themes section.
  4. Click on the "Import Theme" button and select the downloaded JSON file.
  5. Apply the theme and enjoy the mobile-inspired UI on desktop!
